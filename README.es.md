Habilidad de Hechicero de Sintaxis para OpenClaw.

## Qué Soluciona
Transforma automáticamente código legado en patrones modernos sin intervención manual.

## Características
- Activación automática al detectar tareas relevantes
- Resultados profesionales y listos para producción
- Enfoque de seguridad primero
- Soporte para reversión

## Uso
/syntax-sorcerer

## Ejemplo
```
Analizar código para detectar problemas de codificación.
```

## Instalación
Esta habilidad está disponible automáticamente en OpenClaw.
